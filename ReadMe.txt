

I created a new project using Angular7 Web Application(.NET Core2.2) Template in Visual Studio 2019.

*** To Run this app: ***
1. npm install for front-end.
   run "npm install" in FileUploadService\ClientApp.
2. Open FileUploadService.sln.
3. Database Migration.
   run "Update-Database" in "Package Manager Console" of Visual Studio.
4. Run Application
   on Menu, click "Debug"->"Start without Debugging"
   You can see the web browser is opening with "https://localhost:44342/" url.


*** To Run Test ***
1. Front-end test
   run "ng test" in FileUploadService\ClientApp.
2. Back-end test
   In Visual Studio, click "Test"->"Run"->"All Tests".
